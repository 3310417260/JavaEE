## 安装运行
* 本程序为maven项目，所以首先需要下载和配置好maven.http://maven.apache.org/download.cgi
* clone 本项目
* 导入dbfiles\目录下sql文件到数据库
* 配置数据库


### 开发环境
    Ubuntu20.04
    Windows11
    
### 用到的技术	
	JDk8
	Spring
	Spring boot
	Shiro
	Bootstrap3
	JQuery
	Freemarker
	Mybatis
	
### 已实现的功能
* 加密(MD5)
* 文章发表
* 文章查看与修改
* 各种数据统计
